## 🚀 快速开始

### 环境要求
- **Node.js** >= 18.0.0
- **Rust** >= 1.77.0  
- **Windows 10+** (构建目标)

### 安装和运行

```bash
# 1. 进入项目目录
cd dify-client

# 2. 安装前端依赖
npm install

# 3. 开发模式运行
npm run tauri:dev

# 4. Windows 构建
npm run tauri:build
# 或使用提供的构建脚本
./build.bat    # Windows
./build.sh     # Linux/macOS
```
